# bottega-ticket-notifications
A simple nodejs app to help mentors respond to tickets.

### Quick start
Run the command
`npm start`
### Options
* -V, --version             output the version number
* -p, --path <path>         Sets the path of what audio clip will be used (default: "./defaultBeep.ogg")
* -c, --cache <cacheTime>   Set the interval(miliseconds) of the cache clear (default: 600000)
* -P, --player <playerExe>  Sets the player that will play the sound (default: "play")
* -h, --help                output usage information
